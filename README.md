# Ticket Reservation System using Python & MySQL

A console-based application developed to manage ticket bookings and passenger details using Python and MySQL.

## Features
- Create, Read, Update, and Delete (CRUD) operations
- Passenger and ticket management
- MySQL database integration
- Console-based user interface

## Technologies Used
- Programming Language: Python
- Database: MySQL
- Concepts: OOP, CRUD Operations, DBMS

## Learning Outcomes
- Strong understanding of Python database connectivity
- Hands-on experience with MySQL queries
- Improved backend logic and data handling
